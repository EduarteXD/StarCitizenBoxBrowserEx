# 星际公民盒子 浏览器拓展

星际公民盒子浏览器拓展是一个为您将星际公民官网及常用工具网站翻译为中文的浏览器拓展，所有翻译内容来自 星际公民中文社区汉化组。
本插件也内置于 [星际公民盒子](https://jihulab.com/StarCitizenCN_Community/StarCitizenDoctor) 。
